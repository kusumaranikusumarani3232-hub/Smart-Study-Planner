# 📚 Smart Study Planner

A web-based study planner that helps students generate personalized study schedules based on subjects, difficulty, and weak areas.

## 🚀 Features

- 📅 Generate study plan for multiple days
- ⚠️ Prioritize weak subjects
- 📊 Track progress with completion checkbox
- 📈 Visualize study hours using charts
- 💾 Auto-save progress
- 📥 Download plan as CSV

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas

## 💡 How It Works

- User enters subjects and exam duration
- App generates a structured study plan
- Topics are assigned based on subject
- Users mark tasks as completed
- Progress is tracked and saved automatically
